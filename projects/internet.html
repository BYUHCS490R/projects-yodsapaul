Internet

A vast global network that connects to the digital data central. The Internet contains almost all information and resources that are accessible online. The data usually travels through network cables that are hidden uder ground or under the sea.

Web

The information system interface in the Internet that contains different forms of information. Accessible through the Internet. Webs can be found by us finding their destination using links in various forms. For example, HTML, HTTP.

How are the Internet and Web different?

The Internet is the highway that contains a lot of small roads (hyperlink) that connects to different destinations (web) which contain specific information.

ASCII and Unicode

ASCII is the English digital communication that are encoded in the computer which contains different symbols and characters. Meanwhile, Unicode is the same as the ASCII but it has almost every symbols and languages that are avaiable online.

What is UTF-8

UTF-8 is the language format of the Unicode, which are described by numbers. It builds only for computers to understand it to store different language and symbols in the massive database.

URL

The link that has their own unique addresses to locate the information on the Internet.

Domain Name

The Internet address that is readable to people. it usually contains human's language instead of computer's.

HTTP

It stands for Hypertext Transfer Protocol. It is the messenger system for the web. HTTP is in the middle of the communication between web browser and the server.

HTTPS

It is the same thing as HTTP, but it is encrypted each time the data has been exchanged for the sake of securing the information.

IP Address

Shows in nunique numbers that has their own identification to identify the specific device or location.

DNS

The system that converts human-readable domain names into computer languages in order to communicate with the Internet.

W3C Recommendations

Official rules for using Internet and web technologies to help the web works for everyone.

XML, XHTML, HTML, HTML5

1.XML = The language for moving data in a structured way in order to make the data readable for both humans and computers.

2.XHTML = The HTML that follows XML rules. It gives more consistent coding and has easier structures.

3.HTML = HyperText Markup Language. It used to build pages and web. It tells a web browser how to display content on the a web page.

4.HTML5 = Another version of HTML that has interactive and graphic features.
