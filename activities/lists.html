  3. What are special entity characters?

    - Special entity characters are the code versions of symbols that could confuse HTML. Some symbols like <, >, and & have special meaning in web pages, so you write them as entities instead (like &lt;, &gt;, &amp;). This makes sure the browser shows the symbol as text, not as part of HTML code.

  4. What are phrase tags?

    - Phrase tags are HTML tags that mark up small parts of text (a phrase) to give them meaning or special formatting, without creating a new paragraph or block.

  5. What is the difference between OL and UL Tags?

    - Ordered List (OL) makes a list with a specific order so items are numbered/lettered (123 or abc) Use it when they need to be organized or ordered.

    - Unordered List (OL) makes a list without order so things in it use bullets points instead(•).
